# Codex CLI Commands

Command templates for Codex CLI live here.

These should stay close to Codex conventions and avoid assuming Gemini-specific
behavior.

Project-local command templates are not part of the Codex integration path.

Codex projects enter Logos Nous Mode by default after install through
`AGENTS.md`, `.agents/skills/nous/SKILL.md`, step procedures under
`.agents/logos/procedures/`, and `.logos/session/nous-state.json`.
