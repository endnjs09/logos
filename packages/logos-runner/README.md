# logos-runner

Codex CLI target runtime for Logos role workers.

## Commands

- `logos-runner doctor`
- `logos-runner start "<request>"`
- `logos-runner next <plan_id>`
- `logos-runner run-stage <plan_id> <stage>`
- `logos-runner run <plan_id>`
- `logos-runner answer <plan_id> "<answer>"`
- `logos-runner continue <plan_id>`
- `logos-runner execute <plan_id>`
- `logos-runner verify <plan_id>`
- `logos-runner complete <plan_id> <stage>`
- `logos-runner status`
- `logos-runner stages`
