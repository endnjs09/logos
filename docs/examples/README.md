# Examples

This directory is reserved for human-readable examples.

Examples should teach how to author Logos assets without becoming runtime
inputs themselves. Runtime benchmark tasks belong in `benchmarks/`, not here.

## Example Categories

Useful future examples:

- sample `logos.toml`
- sample target manifest
- sample plugin manifest
- sample benchmark task
- sample benchmark suite
- sample run result
- sample measurement log
- sample comparison report
- sample command file
- sample role file
- sample skill file

## Example Rules

Examples should:

- be small
- be realistic
- avoid secrets
- use stable names
- show expected file paths
- explain why fields exist

Examples should not:

- duplicate full reference docs
- include private local paths
- depend on unavailable services
- pretend to be production-ready

## Naming

Use descriptive file names:

```text
sample-logos.toml
sample-target-manifest.toml
sample-plugin-manifest.toml
sample-benchmark-task.yaml
sample-measurement-log.jsonl
```
